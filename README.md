# GCOIN — Official Token Repository

GCOIN is a BEP-20 token deployed on Binance Smart Chain.  
This repository contains the official smart contract, metadata, documentation, whitepaper, security audits, and public website files.

## Contract Address

## Chain
Binance Smart Chain (BEP20)

## Repository Structure
- `/contracts` — Smart contract (Solidity) + ABI  
- `/metadata` — Token metadata for wallets and exchanges  
- `/docs` — Whitepaper, tokenomics, roadmap  
- `/audit` — Basic security audit  
- `/website` — Static website files  
- `/LINKS.md` — Official GCOIN links
