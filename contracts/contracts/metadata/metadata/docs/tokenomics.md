# GCOIN Tokenomics

## Contract
- **Address:** 0x0a94656b2fe1c8f66eebeb45ea5d42854b52a86c  
- **Chain:** Binance Smart Chain (BEP-20)  
- **Decimals:** 18  

---

## Supply Model
- **Mintable:** Yes (owner-controlled)  
- Minting can be disabled permanently by renouncing ownership.  
- Total supply expands only if owner mints more tokens.

- Trading ecosystem

- Community rewards  
- Payment integration (Goopro API)
